API キーの取り方（かんたん3ステップ）

  2. 画面の「APIキーを作成」ボタンをクリックします
  3. 表示されたキー（長い英数字の文字列）をコピーして、下の入力欄に貼り付けます

※ 無料で使えます。Google アカウントがあれば誰でも取得できます。
